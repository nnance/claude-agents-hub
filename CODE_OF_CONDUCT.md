# Code of Conduct
We are committed to fostering a welcoming and inclusive environment for all contributors to the Claude Agents Hub. To ensure that everyone feels safe and respected, we ask all participants to adhere to the following code of conduct:

1. **Be Respectful**: Treat all participants with respect and kindness.
2. **Be Inclusive**: Welcome and include people from all backgrounds and identities.
3. **Be Considerate**: Be mindful of others' feelings and experiences.
4. **Report Issues**: If you encounter any issues or violations of this code of conduct, please report them to the project maintainers.

By following these guidelines, we can create a safe and productive environment for everyone involved in the Claude Agents Hub project.
