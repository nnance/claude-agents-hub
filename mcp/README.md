# MCP Servers

This directory contains the server implementations for the MCP (Modular Control Protocol). Each server is designed to handle specific tasks and functionalities as defined by the MCP specifications.

## Available Servers

**[Fathom MCP Server](./fathom-mcp-servers/README.md)**: A server that integrates with the Fathom API to provide access to meeting recordings, summaries, and transcripts.

## Getting Started
Please refer to the individual server README files for detailed information on installation, configuration, and usage.
